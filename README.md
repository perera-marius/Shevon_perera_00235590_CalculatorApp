Calculator Application (C# Windows Form)

Project Title
Calculator Application – Windows Form App

 Objective
To design and implement a simple calculator using C# Windows Forms that performs basic arithmetic operations — addition, subtraction, multiplication, and division — between two numeric inputs.

Features
- Perform basic arithmetic operations (+, −, ×, ÷)
- User-friendly Windows Form interface
- Error message for division by zero
- Displays results clearly on the same form

 Tools Used
- Microsoft Visual Studio
- C# Programming Language
- .NET Framework (Windows Forms)

How to Run the Application
1. Open the project in **Visual Studio**.
2. Click Run or press F5.
3. Enter two numbers.
4. Click the operation button (+, −, ×, ÷).
5. The result will appear on the screen.
